# Proyecto QA – Pruebas Funcionales Web

## Descripción del proyecto
Este proyecto corresponde a la ejecución de pruebas funcionales manuales sobre una aplicación web para validar la experiencia del usuario, detectar errores y asegurar que el sistema cumpla con los requisitos definidos.

## Objetivo
Validar el correcto funcionamiento de:
- formularios
- navegación
- mensajes del sistema
- validaciones de campos
- comportamiento visual
- flujo del usuario

## Herramientas utilizadas
- Jira
- Google Sheets
- Navegadores web
- Documentación QA

## Actividades realizadas
- Ejecución de pruebas funcionales
- Pruebas exploratorias
- Validación de interfaz de usuario
- Verificación de mensajes de error
- Documentación de defectos
- Priorización de incidencias

## Escenarios validados
Se probaron funcionalidades como:
- inicio de sesión
- registro de usuarios
- recuperación de contraseña
- validación de formularios
- navegación entre módulos
- mensajes de confirmación

## Resultados obtenidos
- Se ejecutaron más de 25 pruebas manuales
- Se identificaron múltiples defectos funcionales
- Se mejoró la cobertura de validación del sistema
- Se documentaron errores para facilitar la corrección del equipo técnico

## Habilidades demostradas
- Atención al detalle
- Pensamiento analítico
- Comunicación de hallazgos
- Organización de pruebas
- Validación funcional
